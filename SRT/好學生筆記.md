# 好學生筆記 (Good Student Notes)

> 將課程教材轉化為帶有專業視角的手寫風格筆記圖片

---

## 🎯 核心理念

作為一個「好學生」，在聽講時的重點**不在於抄寫每一個字**，而在於：

| 筆記層次     | 思考問題                 | 筆記動作             |
| ------------ | ------------------------ | -------------------- |
| **核心定義** | 這是什麼？               | 圈出關鍵字、標註定義 |
| **連結已知** | 這跟我之前學的什麼有關？ | 加入專業類比         |
| **提煉觀點** | 講者覺得這像什麼？       | 整理講者的隱喻與比喻 |
| **預判問題** | 為什麼我們要學這個？     | 寫下疑問與延伸思考   |

---

## 🔄 執行流程（三步驟）

### Step 1️⃣ 詢問學習困難範圍

```
📖 請問你在課程的哪個部分感到困難或想深入理解？

你可以告訴我：
- 主題名稱（例如：「Sparse Autoencoder」「9.11問題」）
- 頁碼或段落範圍
- 關鍵字

我會從逐字稿中找到對應內容。
```

### Step 2️⃣ 詢問專業背景

```
🎓 你是哪種專業背景的「好學生」？

例如：
- 會計師 / 財務人員
- 建築設計師 / 室內設計師
- 軟體工程師 / 資料科學家
- 醫療專業人員
- 教育工作者
- 製造業工程師
- 法律專業
- ...其他專業

這將決定筆記中的類比角度與補充觀點。
```

⚠️ **必須收集以上兩項資訊後，才能啟動好學生筆記計劃。**

### Step 3️⃣ 啟動好學生筆記（兩階段生成）

#### 階段 A：生成「原始課程講義」圖片
- 從逐字稿中截取指定範圍的文字
- 生成乾淨、專業的教材截圖（純文字，無註解）
- 這是「底圖」

#### 階段 B：疊加「好學生筆記」
- 以原始講義圖片為底
- 根據專業背景，加上手寫風格的註解、類比、流程圖
- 生成最終的「好學生筆記」圖片

## 🎨 視覺設計規範

### 顏色系統

| 顏色       | 用途                            | 色碼               |
| ---------- | ------------------------------- | ------------------ |
| 🔵 **藍色** | 重點摘要、關鍵字標註、圈選      | `#1976D2`          |
| 🔴 **紅色** | 思考疑問、Insight、Bug/問題標記 | `#D32F2F`          |
| 🟠 **橘色** | 專業類比、領域連結              | `#E65100`          |
| 🟢 **綠色** | 專業人設的補充說明              | `#388E3C`          |
| ⬛ **黑色** | 原始課文內容                    | `#333333`          |
| 🟡 **黃色** | 螢光筆標記重點                  | `#FFEB3B` (半透明) |

### 手寫風格元素

- **底線標記**: 重要詞彙下方的波浪線或直線
- **圈選標記**: 關鍵概念的圓圈或橢圓框
- **箭頭連結**: 概念間的邏輯關係
- **邊欄註解**: 頁面右側或左側的補充說明
- **流程圖**: 簡化的概念流程示意
- **驚嘆號/問號**: 「！」或「？」的思考標記
- **Post-it 便利貼**: 專業類比區塊

## 📝 Prompt 設計模板

### 階段 A：生成原始課程講義

```markdown
生成一張乾淨的課程講義圖片，使用繁體中文。

## 內容
標題：{章節標題}

內文：
{從逐字稿截取的原始課文}

## 視覺風格
- 乾淨的白色/淺灰色背景
- 專業的印刷字體
- 良好的段落間距
- 類似教科書或投影片截圖
- 不要任何手寫註解或裝飾
```

### 階段 B：生成好學生筆記

```markdown
將這張課程講義圖片轉換為「好學生筆記」版本。
保留原始印刷文字作為底層，在上面加上手寫風格的註解。

## 學生背景
這位學生是 {使用者專業}，請用該專業的術語和經驗來做類比。

## 註解規則

### 藍色筆跡（重點標記）
- 圈選關鍵術語
- 底線重要句子
- 框選核心概念

### 紅色筆跡（疑問與洞察）
- 在驚訝的概念旁加「！」
- 寫下疑問「？」
- 底部核心洞察

### 橘色/綠色筆跡（專業類比）
- 邊欄加入專業術語對照
- 用 {使用者專業} 的角度重新詮釋概念
- 簡單的流程圖/示意圖

### 底部總結框
- 用一句話總結核心洞察
- 連結到 {使用者專業} 的實際應用

## 視覺風格
- 原始印刷文字保持可見
- 手寫註解疊加在上方
- 稍微凌亂但有條理的手寫字
- 螢光筆效果標記重點
- 箭頭連接概念
- Post-it 便利貼風格的類比區塊
```

## ⚠️ 注意事項

1. **必須先確認兩項資訊**：學習困難範圍 + 專業背景
2. **兩階段生成**：先生成乾淨底圖，再疊加手寫註解
3. **保持原文完整性**：筆記是「補充」而非「取代」原始內容
4. **類比要合理**：確保專業類比在邏輯上是正確且有意義的
5. **視覺清晰度**：圖片解析度建議至少 150 DPI
6. **中文字型**：使用支援繁體中文的字型
